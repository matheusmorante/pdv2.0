# Regras do Usuário - Antigravity Kit

Este arquivo contém as regras globais e específicas para o projeto, conforme solicitado pelo usuário.

## Regras Globais

1.  **Código Limpo:** Siga princípios de código limpo e modularizado.
2.  **Documentação de Ideias:** Sempre salve minhas ideias e planos pendentes em um arquivo para posterior consulta e lembrete.
3.  **Economia de Cota:** Economize o máximo de cota que puder do plano de assinatura.
4.  **Ambientes de Desenvolvimento e Produção:** Lembre que tem o ambiente de desenvolvimento e produção, deixe ajustado para cada um o que for fazer. Sempre considere ambos ao configurar rotas, variáveis de ambiente e outros dados ambíguos.
5.  **Relato de Bugs:** Relate imediatamente bugs que estão te impedindo para eu saber da demora enquanto você continua fazendo o que está fazendo.
6.  **Esclarecimento de Dúvidas:** Me pergunte se tiver dúvida de ambiguidade ou entendimento no que eu disser ou mostrar.
7.  **Persistência de Regras:** Salve os rules sempre em um arquivo do projeto.

## Frontmatter GEMINI.md

Este projeto utiliza o kit Antigravity com os seguintes protocolos:

-   **MANDATÓRIO:** Ler o arquivo do agente apropriado e suas skills ANTES de realizar qualquer implementação.
-   **Enforcement Protocol:** Ler -> Entender -> Aplicar.
-   **Socratic Gate:** Para requisições complexas, PARE e PERGUNTE primeiro (mínimo 3 perguntas estratégicas).
-   **Final Checklist:** Uma tarefa não está terminada até que o `checklist.py` retorne sucesso.
